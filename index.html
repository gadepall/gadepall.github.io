About Us

Center Justified Text

Narayanpal was established in 2024 by Dr. GVV Sharma, faculty in the EE department, IIT Hyderabad, to promote semiconductor and wireless/ML/signal processing technologies in a hands on fashion. This is done by using Dashak: The Decimal Game, which is a microcontroller kit that was designed by Dr. Sharma to teach digital design to students at IITH. Through the microcontroller boards (Arduino/Vaman), students learn how to code using verilog, assembly and embedded C, which are mandatory skills for all hardware engineers. Further, students learn how to do scientific computing using hardware, which is essential for all engineering, including wireless and machine learning applications. The key feature is the use of free software and low cost hardware, with android phones being used for programming. This democratizes engineering education by allowing students in remote areas to compete on a global scale. This approach to pedagogy was developed as part of the fractal academic program at IIT Hyderabad, which reduces learning time by 50%, thus meeting many objectives of the New Education Policy, 2020.
